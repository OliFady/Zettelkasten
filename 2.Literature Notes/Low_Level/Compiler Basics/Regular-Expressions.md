# Regular Expressions

## Automata 
a -> b -> c -> f 

- Automata is equivalent to Regexs
- A Character is a Regex
- a | b transitions to a branch
- a* can skip an Automata
- a+ is aa*
- a{3} is aaa
- [abc] is a|b|c
- [a-zA-Z]
- [^abc] inverts the meaning
- . matches anything [^] 
- .*foo.* finds a foo inside a string
- ^ is start of the stream
- $ is end of the stream
